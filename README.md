# Backend

## Деплой
```bash
cd backend
make
```

## Запуск сервера разработки

```bash
cd backend
make run
```

## Генерация рандомных данных
```bash
cd backend
make generate
```

# Frontend

## Деплой
```bash
cd frontend/ng-prj
npm install
```

## Запуск сервера разработки
```bash
ng serve
```
